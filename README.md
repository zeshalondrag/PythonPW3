Практическая работа №3 «Доработка игры – работа над сохранениями»
-----------------------------------------------------------------
Задача состоит в усовершенствовании игры, разработанной на предыдущей практике, путем внесения функционала сохранения. Требуется сохранять выходные данные игры и добавить возможность удаления сохраненных состояний. Вся информация должна храниться в формате JSON. Кроме того, необходимо реализовать автоматическую запись данных в CSV файл после каждого завершения игры, чтобы в файле присутствовали все пользователи с соответствующей информацией
